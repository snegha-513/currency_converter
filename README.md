# Currency Converter App (Tkinter GUI)

## Description
This project is a Python-based Currency Converter application developed using Tkinter as part of the RISE 3.0 Internship Program by Tamizhan Skills.

## Features
- GUI-based application
- Dropdown selection for currencies
- Static exchange rates
- Easy to use interface

## Technologies Used
- Python
- Tkinter

## How to Run
1. Install Python 3.x
2. Run the application using:
   python currency_converter_gui.py

## Input
- Amount
- Source Currency
- Target Currency

## Output
- Converted currency value displayed on screen
